# Sensihi Website Copilot

AI-powered website copilot built with:

- MCP-style orchestration
- Vector search (Supabase)
- Session memory
- Lead scoring
- Framer UI

## Deploy

1. Add env vars (see `.env.example`)
2. Deploy to Vercel
3. Connect to Framer

## Production Ready

Yes.

- **Backend (Edge, secure)** — Vercel Edge API
- **Vector search** — Supabase + OpenAI embeddings
- **Memory** — In-session conversation history
- **Lead intelligence** — Intent + scoring
- **Analytics** — Event logging
- **Framer-native UI** — Paste `framer/SensihiCopilot.tsx` as a Code Component

This is not a demo — this is something you can sell, scale, and showcase.
